# mip-jx-ad

mip-jx-ad 第三方聚效广告扩展组件

标题|内容
----|----
类型|通用，定制
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://mipcache.bdstatic.com/static/v1/mip-jx-ad/mip-jx-ad.js

## 示例

用于投放聚效第三方广告的扩展组件
```html
<mip-jx-ad 
     type="ad-qwang"
     unionId="u2816363"
	 cpro_pswidth="309"
	 cpro_psheight="250"
     >
</mip-jx-ad>
```
