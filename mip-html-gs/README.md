# mip-html-gs
mip-html-gs 下载站定制高速下载组件

标题|内容
----|----
类型|通用
支持布局|N/S
所需脚本|https://mipcache.bdstatic.com/static/v1/mip-html-gs/mip-html-gs.js   

## 示例

### 标签使用示例
```html
<mip-html-gs Qi-url="//www.***.com/ajax/" Qi-arr='{"id":"123"}'></mip-html-gs>
```

## 属性

### Qi-url

说明：AJAX请求URL  
必选项：是  

### Qi-arr

说明：配置数组  
必选项：是
