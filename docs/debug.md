调试 MIP 扩展组件
==============

我们提供了 `mip-cli` 工具，可以帮助你在开发 MIP 扩展组件时进行调试。


安装调试工具
-----

想要安装 `mip-cli` 工具，我们需要通过 `npm install -g` 命令。

```
$ [sudo] npm install -g mip-cli
```

安装完成后，我们将获得一个 `mip` 命令：

```
$ mip
bla bla
```

直接从官方 npm registry 安装，可能会由于网络原因，导致安装时间较长或安装失败。此时我们可以选择速度更快的 registry。

```
$ ........
```
