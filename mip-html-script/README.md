# mip-html-script
mip-html-script 加载JS执行方法

标题|内容
----|----
类型|通用
支持布局|N/S
所需脚本|https://mipcache.bdstatic.com/static/v1/mip-html-script/mip-html-script.js   

## 示例

### 标签使用示例
```html
<mip-html-script Qi-src="Qi.js" Qi-call="Qi.init()"></mip-html-script>
```

## 属性

### Qi-src

说明：JS链接  
必选项：是  

### Qi-call

说明：JS方法名称  
必选项：是
